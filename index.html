PeerGaming - Share the Fun

A Client-Side Multiplayer Framework using WebRTC

About

With the rising of WebRTC, the possibilities for modern browser based games are changing. Besides new interactions through inputs like sound & audio - P2P connections are possible and allow direct communication with other players.

PeerGaming is a Multiplayer Gaming Framework for the Web, which uses standard web technologies to connect multiple Clients. It allows you to create your own Multiplayer Game, without concerning an additional setup like a server (*).

* an example for the initial process of bootstraping & brookering will be be provided
